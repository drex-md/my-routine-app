Setup Email OTP
To send OTP emails, this app uses EmailJS (free). You only set this up once.
Step 1
Go to emailjs.com and create a free account.
Step 2
Go to Email Services and connect your Gmail or any email. Copy the Service ID.
Step 3
Go to Email Templates and create a new template. Set Subject as Your OTP Code and body as:

Your verification code is: {{otp_code}}
Valid for 5 minutes.
Copy the Template ID.
Step 4
Go to Account and copy your Public Key.

🔥 Connect Firebase
Firebase is a free Google cloud service. Your routine data will be stored safely online — accessible from any device, anytime. Setup takes 5 minutes and you only do it once.
Step 1
Go to console.firebase.google.com and sign in with your Google account. Click Add project, name it MyRoutine, disable Analytics, click Create project.
Step 2
In your project dashboard, click the Web icon (</>). Register with any nickname. You will see a firebaseConfig block with values — keep this page open.
Step 3
In the left sidebar go to Build → Realtime Database → Create database. Choose your region. Select Start in test mode. Click Enable.
Step 4
Copy each value from your firebaseConfig and paste below:
API Key
Auth Domain
Database URL
Project ID
